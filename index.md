## **Hello in Python vs Java vs C++**
Python:
```
print("Hello World!")
```

Java:
```
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World");
    }
}
```

C++:
```
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World!";
    return 0;
}
```
